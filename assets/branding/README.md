# axis branding

Primary draft assets for the `axis` identity.

Files:

- `axis-icon.svg`: app icon / badge. Dark spatial grid, central axis node, stylized `A`.
- `axis-logo.svg`: horizontal lockup with wordmark for docs or landing pages.
- `axis-brand-sheet.svg`: square presentation sheet for quick previews.

Concept:

- the grid and crossing guide lines signal a spatial workdesk;
- the warm center node is the active focus point;
- the green marker hints at attention routing across multiple live contexts.

Palette is aligned with the current `axis-app` UI accents:

- navy base: `#091016` to `#131A20`
- grid/axis: `#182028`, `#365066`
- shell/attention accents: `#E59A49`, `#77D19A`, `#7CC7FF`
- wordmark light: `#F7EFE5`
